# Getting Started with Modding

Welcome! This guide will help you understand the basics of modding and get you started on your creative journey.

## Prerequisites

Before you begin, you'll need:
- Basic familiarity with file systems and folders
- A text editor (VS Code, Sublime Text, etc.)
- The game's modding tools (check the Tools section)

## What is Modding?

Modding involves creating modifications to games by:
- Editing game assets (characters, textures, models)
- Creating new content
- Modifying game behavior

## Key Concepts

### Asset Extraction
Learn how to extract game assets for editing:
1. Use the appropriate extraction tool
2. Organize assets into folders
3. Keep track of file formats

### Asset Packing
Once you've modified assets, you need to pack them back:
1. Ensure your modifications are compatible
2. Use validation tools
3. Test thoroughly

## Common Questions

**Q: Do I need programming knowledge?**
A: No! Most modding tasks use visual editors and drag-and-drop tools.

**Q: Can I share my mods?**
A: Yes! Check community guidelines for distribution and licensing.

**Q: Where can I get help?**
A: Check the [Community page](/community/) for resources and discussion.
