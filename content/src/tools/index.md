# Modding Tools

A curated collection of tools for game modding. Each tool includes documentation, guides, and examples.

## Available Tools

- **znelchar** - Character extraction, editing, and packing tool
  - Extract character data to editable formats
  - Validate and verify modifications
  - Pack modified data back into game files
  
  [View znelchar Documentation →](/tools/znelchar/)

## Adding New Tools

To add tool documentation to this site:

1. Add a `docs/` folder to your tool's directory
2. Include a `README.md` with overview and usage
3. Add setup instructions and examples
4. The tool docs will auto-appear here on next build

See [CONTRIBUTING.md](/CONTRIBUTING.md#adding-tool-documentation) for details.
